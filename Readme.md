# 🎯 Jogo da Forca em C++

![C++](https://img.shields.io/badge/c++-%2300599C.svg?style=for-the-badge&logo=c%2B%2B&logoColor=white)

Implementação simples do clássico jogo da forca utilizando C++ e execução via terminal.

O projeto foi desenvolvido com foco em praticar:
- Manipulação de arrays
- Estruturas de repetição
- Estruturas condicionais
- Controle de fluxo
- Portabilidade básica entre Windows e Linux

## 🖥️ Demonstração

O jogo funciona em modo terminal:

1. Um jogador digita a palavra secreta.
2. A tela é limpa.
3. O segundo jogador tenta adivinhar a palavra.
4. O jogador possui 6 chances para errar.

## 🧠 Conceitos aplicados

- Arrays de caracteres
- Controle manual de tamanho de string
- Laço while para controle de jogo
- Uso de diretivas de pré-processador (#ifdef _WIN32)
- Uso de system("cls") e system("clear")
